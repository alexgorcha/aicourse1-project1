# DEAD DROP Landing Page

Одностраничный промо-лендинг для вымышленной игры **DEAD DROP** — cinematic solo extraction FPS в заброшенном неоновом мегаполисе.

## Цели

- собирать заявки в публичный плейтест;
- подталкивать игроков к добавлению игры в wishlist Steam;
- презентовать игру через атмосферу, ключевые механики и единый набор игровых скриншотов.

## Стек

- HTML
- CSS
- JavaScript

Проект не требует сборки и подходит для публикации через GitHub Pages.

## Структура

```text
.
├── index.html
├── styles.css
├── script.js
└── assets/
    ├── image-prompts.md
    └── images/
```

## Локальный запуск

Можно открыть `index.html` напрямую в браузере или поднять простой локальный сервер:

```bash
python -m http.server 4173
```

После этого сайт будет доступен по адресу:

```text
http://localhost:4173
```

## Контент

- главный CTA: `Join the Playtest`;
- вторичный CTA: `Wishlist on Steam`;
- CTA сейчас ведут на якоря-заглушки;
- изображения и промпты для их генерации лежат в `assets/`.

## Публикация

Для GitHub Pages достаточно выбрать ветку `main` и корень репозитория как источник публикации в настройках GitHub Pages.
